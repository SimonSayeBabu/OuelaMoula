# OuélaMoula

OuélaMoula est une carte qui affiche la position des Milliardaires a travers le monde et affiches diverse informations sur chaqu'un.

## Prérequis

Necessite Java 25, telecharger le via le [lien officiel](https://www.oracle.com/java/technologies/downloads/).

## Demarrage

```bash
java ./JExpress.java
```

Puis acceder à : 
```
http://localhost:8080/main.html
```
